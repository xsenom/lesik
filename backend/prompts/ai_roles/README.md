# AI роли: куда класть инструкции

Файлы инструкций для ИИ-ролей должны лежать в папке:

`backend/prompts/ai_roles/`

Используемые имена файлов:
- `01_Архитектор_позиционирования_и_ЦА.txt`
- `02_Стратег_продвижения_в_соцсетях.txt`
- `03_Продуктовый_архитектор.txt`
- `04_Ежедневный_менеджер_продвижения.txt`
- `05_Маркетинговый_офис_10_сотрудников.txt`

Бэкенд автоматически читает эти файлы и подставляет их как системную инструкцию
в зависимости от выбранного режима `agent`.

## Логика
1. Архитектор позиционирования и ЦА
2. Продуктовый архитектор
3. Стратег продвижения в соцсетях
4. Ежедневный менеджер продвижения
5. Маркетинговый офис 10 сотрудников
